# BISOFT-15 Diseño y Construcción de Componentes
**Universidad CENFOTEC** | Bachillerato en Ingeniería de Software

Portafolio del curso BISOFT-15. Cubre principios, técnicas y
tecnologías para construir componentes de software en Java,
incluyendo diseño, modularidad, encapsulación y reutilización.

## Contenido
- `semana-01/` — El concepto de componente
- `practicas/` — Prácticas en clase
- `estudios-de-caso/` — Estudios de casos
- `proyecto/` — Proyecto de aplicación de software

## Temas del curso
1. El concepto de componente
2. Diseño de componentes
3. Proceso de desarrollo con componentes
4. Los controles como componentes
5. Plataformas de componentes
6. Modelos de componentes
