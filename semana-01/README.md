# Semana 1 — El Concepto de Componente

## ¿Qué es un componente?
Unidad modular y reutilizable de software que encapsula una
funcionalidad específica. Funciona de forma independiente y
puede ensamblarse con otros componentes.

## Características clave
- **Encapsulación:** oculta el "cómo", expone el "qué"
- **Reutilización:** usable en múltiples contextos sin reescribirse
- **Modularidad:** unidades independientes combinables
- **Interoperabilidad:** se comunica con otros mediante interfaces

## Analogía
Como piezas de LEGO: cada una tiene forma, propósito e interfaz
de conexión, y sigue un estándar.
