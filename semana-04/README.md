## Semana 4 — Componente de Bitácora de Errores con MongoDB

Laboratorio: componente Java reutilizable que registra y consulta
errores mediante una API REST que almacena en MongoDB.

**Tecnologías:** Java 21, HttpClient, MongoDB Atlas (vía API REST)

**Colección:** erroresIsaMurillo

**Componente:** model / service / util → empaquetado como .jar
**Consumidor:** proyecto independiente que importa el .jar
